# JS Web Scraper

A simple Node.js web scraper that extracts content from a public website and saves it as JSON.

## Installation

```bash
npm install
```
Usage
```
node scraper.js
```

Output
The scraper will save results to output.json.

Dependencies

axios
cheerio
